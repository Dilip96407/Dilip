User interface

The components in details:

Menu bar with pull down menus

Toolbar, a function bar with smart buttons

Navigation bar. A "folder bar" with various general possibilities to start the application. All entries that you find in the navigation bar can be found as well in the menu bar. Let us call it a menu bar with "web look and feel".

Quicksearch. This component helps you to get a quick access to certain data, such as clients, contacts or colleagues.

Working area. In this area "runs" the application. Here all the information is displayed.

Status bar. In the status bar you get application-specific hints.

You can switch on and off the navigation bar, the toolbar and the status bar via the menu bar. If the navigation bar is deactivated the quicksearch function automatically is turned off.
